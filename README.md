Mini CRM Distribution Service

Микросервис для автоматического распределения лидов между операторами по разным каналам на Python с использованием FastAPI и SQLAlchemy.

Функционал:

Поддерживает хранение операторов и их нагрузку.

Ведёт учёт лидов и каналов поступления заявок.

Автоматически назначает подходящие операторы новым заявкам с соблюдением лимитов и компетенций.

Особенности:

Разделение обязанностей операторов и источников (ботов).

Гибкая настройка распределения на основе числовых весов.

Оптимизированное распределение заявок по ограничениям нагрузки операторов.

Запуск:

Установка зависимостей:

pip install -r requirements.txt

Запуск приложения:

uvicorn main:app --reload

API-документация через Swagger UI:

http://127.0.0.1:8000/docs

Маршруты API:

Метод /Эндапоинт/ Назначение

POST	/create_operator/	Создание нового оператора

GET	/list_operators/	Получение списка операторов

PUT	/update_operator/{operator_id}	Обновление данных оператора

POST	/register_contact/	Регистрация нового обращения

GET	/view_contacts/	Просмотр всех зарегистрированных обращений

Модель данных:

Три ключевые компонента:

Операторы: отвечают за обработку обращений.

Лиды: представляют конечных клиентов.

Источники: каналы поступления заявок (боты).

Связи между ними представлены в таблице SQLAlchemy в файле models.py.
